== Description ==

= Adaire Blocks: Pay Only for What You Use (TM) =

Most page builders make you pay for dozens of widgets you may never touch. Adaire Blocks takes a smarter approach.

Start with a powerful free collection of Gutenberg blocks for layout, content, social proof, media, and animation. When your site needs more, unlock only the blocks you actually want. No bloated bundles. No paying for 100 features just to use five.

That is the Adaire Blocks philosophy: modern design tools, native Gutenberg workflows, and flexible upgrades that grow with your site.

Adaire Blocks is designed for site owners, freelancers, and agencies who want polished pages inside the native WordPress editor. Build landing pages, service pages, portfolios, testimonials, blog layouts, and promotional sections without switching to a separate builder experience.

== Installation ==

= Minimum requirements =

* WordPress 6.7 or greater
* PHP 7.4 or greater

= Recommended requirements =

* PHP 8.0 or greater
* A modern block theme or block-compatible WordPress theme
* Adequate hosting resources for media-rich, animation-heavy pages

= Getting started =

1. Install Adaire Blocks from the WordPress plugin repository, or upload the plugin files to the `/wp-content/plugins/adaire-blocks` directory.
2. Activate the plugin from the `Plugins` screen in WordPress.
3. Open any page or post in the block editor.
4. Click the inserter and search for `Adaire`, or browse the available blocks in your block library.
5. Add the blocks you want, then customize layout, colors, typography, spacing, and animations in the editor sidebar.
6. Publish or update the page to make your new section live.

For help, product updates, or more information, visit https://adaire.digital/
