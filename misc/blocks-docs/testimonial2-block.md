# Testimonial 2 Block Documentation

## Overview
**Testimonial 2** is a bold, image-forward testimonial layout that pairs a large portrait with a floating testimonial card, company logo, and navigation arrows. It is ideal for long-form quotes, founder stories, or hero testimonials where photography and brand logos take center stage.

## Features
- **Carousel Engine** – Horizontal slider with keyboard-accessible navigation buttons.
- **Portrait & Logo Support** – Separate media pickers for the person image and company logo, each with responsive sizing.
- **Typography Controls** – Independent font size/weight controls for the quote, person name, and job title (desktop/tablet/mobile).
- **Color System** – Set background, card background, quote accent image opacity, description color, name/title colors, and navigation button fills.
- **Layout Controls** – Container mode (full or constrained), responsive max-width sliders, spacing between cards, and carousel speed.

## How to Use
1. Insert “Testimonial 2” from the block inserter.
2. Replace the demo testimonial with your own copy, portrait, and logo.
3. Use the “Add Testimonial” button in the sidebar to append additional slides.
4. Adjust navigation button colors/size and quote mark opacity to match your brand.
5. Preview on desktop, tablet, and mobile to confirm responsive typography.

## Navigation & Accessibility
- Navigation buttons are real `<button>` elements with focus rings.
- Keyboard controls: Tab to focus the arrows, Enter/Space to activate.
- Carousel speed (ms) and space-between (px) settings determine the slide animation.

## Tips
- Use high-resolution portraits with similar aspect ratios for consistent layout.
- Keep testimonial copy concise (~2–4 sentences) for readability.
- If the background color is bright, adjust description/name/title colors for contrast.
- Logos look best when uploaded with transparent backgrounds (PNG/SVG).
