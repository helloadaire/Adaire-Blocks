# Adaire Blocks v1.1.7 Release Notes

## New Blocks - Animation on Scroll, Video Player, and Maps

This release introduces three powerful new blocks: **Animation on Scroll Block**, **Video Player Block**, and **Map Block**, giving you complete control over scroll animations, video embedding, and interactive location displays.

### Animation on Scroll Block - Complete Control Over Scroll Animations

The **Animation on Scroll Block** gives you complete control over scroll-triggered animations, with configurable distance, flip options, rotate animations, and improved threshold handling.

### Major New Features

#### Video Player Block
- **Dual Platform Support**: Embed videos from YouTube or Vimeo with seamless switching
- **Smart URL Parsing**: Automatically extracts video IDs from various URL formats (watch, embed, short links, etc.)
- **Flexible Video Controls**: Configure autoplay, mute, controls visibility, and loop settings
- **Responsive Container**: Full width or constrained container with responsive max-width controls
- **Customizable Height**: Set container height with px, rem, or vw units
- **Border Radius Control**: Adjustable border radius (0-100px) for rounded corners
- **Responsive Spacing**: Independent margin and padding controls for desktop, tablet, and mobile
- **Wide & Full Alignment**: Supports wide and full width alignments

#### Map Block
- **Multiple Location Support**: Display multiple locations with interactive navigation
- **Google Maps Integration**: Embed Google Maps with custom embed URLs
- **Leaflet Fly Mode**: Optional single map mode with smooth fly animations between locations using Leaflet
- **Coordinate-Based Navigation**: Set latitude/longitude coordinates for precise map positioning
- **Location Management**: Add, remove, and reorder locations with country, address, phone, and email
- **Customizable Appearance**: Control navigation text color, active background, active text color, and font weight
- **Configurable Fly Speed**: Adjust animation duration for map transitions (0.2-3 seconds)
- **Responsive Design**: Full support for mobile, tablet, and desktop with responsive container and spacing
- **Two Display Modes**: Choose between single map fly mode or multiple map display

#### Animation on Scroll Block
- **Complete Animation Control**: Create scroll-triggered animations with extensive customization options
- **12 Animation Types**: Choose from fade-in, fade-out, fade-left, fade-right, fly-up, fly-down, fly-left, fly-right, grow, shrink, bounce, flip, rotate, blur-in, and blur-out
- **Configurable Distance**: Adjust animation distance for fly and fade animations (0-300px)
- **Flip Animation Options**: Control flip axis (horizontal/vertical) and direction (clockwise/anti-clockwise)
- **Rotate Animation**: New rotate animation type with configurable easing and duration
- **Live Editor Preview**: Preview animations in real-time as you adjust settings
- **Bidirectional Reversal**: Animations reverse when scrolling back up or down past threshold
- **Smart Threshold Handling**: Prevents animation conflicts with delay settings

### Getting Started

#### Animation on Scroll Block

1. **Add the Block**: Insert "Animation on Scroll" from the block inserter
2. **Choose Animation**: Select your desired animation type from the dropdown
3. **Configure Settings**: Adjust duration, delay, easing, and other options
4. **Set Distance**: For fly/fade animations, set the distance (for flip, set axis/direction)
5. **Add Content**: Add any content blocks inside the animation block
6. **Preview**: See your animation in the editor or on the frontend
7. **Responsive**: Adjust settings for different device sizes if needed

#### Video Player Block

1. **Add the Block**: Insert "Video Player Block" from the block inserter
2. **Choose Platform**: Select YouTube or Vimeo
3. **Enter Video URL**: Paste your video URL (or video ID) - the block will automatically parse it
4. **Configure Controls**: Set autoplay, mute, controls, and loop options
5. **Customize Container**: Adjust height, border radius, and container mode
6. **Responsive Settings**: Configure margins, padding, and max-width for different devices
7. **Preview**: See your video in the editor and on the frontend

#### Map Block

1. **Add the Block**: Insert "Map Block" from the block inserter
2. **Add Locations**: Click "Add Location" to create your first location
3. **Enter Location Details**: Fill in country, address lines, phone, email, and map embed URL
4. **Set Coordinates** (Optional): Add latitude/longitude for fly mode with precise positioning
5. **Choose Display Mode**: Enable "Single map fly mode" for smooth transitions or use multiple maps
6. **Customize Appearance**: Adjust navigation colors, font weight, and fly speed
7. **Manage Locations**: Add, remove, or reorder locations as needed
8. **Responsive Settings**: Configure container and spacing for different device sizes
