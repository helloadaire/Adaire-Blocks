# Adaire Blocks v1.2.1 – Maintenance Update

This is a maintenance release that improves the underlying icon management system and build process.

## What's Improved

### Better Icon Management
The plugin now uses a unified icon mapping system that makes it easier to maintain and update block icons. This improvement ensures consistent icon display across the editor, block management page, and documentation.

### Build Process Enhancements
The build scripts have been updated to use a centralized configuration system, making future updates and maintenance more streamlined.

## For Users

This release doesn't add any new features or blocks, but it improves the stability and maintainability of the plugin. All existing blocks continue to work exactly as before.

## Technical Details

- Improved icon management with centralized configuration
- Enhanced build scripts for better consistency
- Fixed icon path corrections for proper tier alignment

No action required—just update and enjoy the improved underlying systems!
