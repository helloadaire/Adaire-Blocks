# Adaire Blocks v1.1.4 Release Notes

## Mega Menu Block Enhancements & Limit Management Improvements

This release focuses on significant improvements to the Mega Menu Block's limit management system and scroll-based opacity functionality, providing better user experience and more responsive interactions.

### Major Improvements

#### Mega Menu Block Limit Management
- **Dynamic Limit System**: Replaced hardcoded menu item limits with a proper `useBlockLimits` hook implementation
- **Premium/Free Tier Integration**: Mega Menu now properly respects the plugin's premium/free tier configuration
- **Consistent User Experience**: Unified limit management system across all blocks for better consistency
- **Proper Upgrade Notices**: Enhanced upgrade messaging with `UpgradeNotice` component integration

#### Scroll-Based Opacity Enhancement
- **Improved Responsiveness**: Fixed "Increase Opacity on Scroll" functionality with immediate response (removed scrub delay)
- **Faster Transition**: Reduced scroll distance from 200px to 50px for quicker opacity changes
- **CSS Compatibility**: Added proper CSS rules to allow JavaScript to control background opacity
- **Better Performance**: Optimized scroll trigger implementation for smoother animations

### Technical Improvements

#### Code Quality & Architecture
- **Hook Integration**: Properly integrated `useBlockLimits` hook for dynamic limit checking
- **Component Consistency**: Aligned Mega Menu with other blocks' limit management patterns
- **CSS Override Resolution**: Fixed CSS conflicts that were preventing opacity changes
- **Performance Optimization**: Streamlined JavaScript code for better scroll performance

#### User Interface Enhancements
- **Immediate Feedback**: Opacity changes now respond instantly to scroll events
- **Visual Consistency**: Unified upgrade notice styling across all blocks
- **Better Error Handling**: Improved validation and error handling for limit checks

### Bug Fixes

- **Mega Menu Limits**: Fixed hardcoded `FREE_TIER_ITEM_LIMIT = 5` that wasn't respecting premium/free tier settings
- **Opacity Scroll**: Resolved CSS override issues preventing background opacity changes
- **Limit Validation**: Fixed `addMenuItem` function to properly check limits using the hook system
- **Upgrade Messaging**: Corrected upgrade notice display logic for better user guidance

### What's Fixed

- **Mega Menu Block**: Now properly uses the plugin's limit management system instead of hardcoded limits
- **Scroll Opacity**: "Increase Opacity on Scroll" feature now works correctly with immediate response
- **Premium Integration**: Mega Menu limits now properly respect premium/free tier configuration
- **User Experience**: Consistent limit management and upgrade notices across all blocks

### Benefits for Users
