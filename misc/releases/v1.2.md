# Adaire Blocks v1.2.0 Release Notes

Version 1.2.0 is a feature release that introduces five powerful new blocks across Free, Plus, and Premium tiers, along with significant improvements to the block management interface and icon system.

## New Blocks

### Flip Card Block (Plus)
- Interactive flip card system with separate front and back content areas using inner blocks
- Smooth 3D CSS transitions with customizable flip direction and timing
- Hover-triggered flip animations with optional click-to-flip support
- Full inner blocks support allowing any Gutenberg blocks on both sides
- Responsive width controls for desktop, tablet, and mobile breakpoints
- Customizable card styling including padding, border radius, and hover effects
- Child blocks: Flip Card Front and Flip Card Back for structured content management

### Pricing Table Block (Plus)
- Responsive pricing table layout with CSS grid system
- Monthly/yearly billing toggle with automatic price switching
- Customizable pricing cards with add/remove/reorder functionality
- Per-card feature lists with add/remove controls and rich text support
- Individual card styling including padding (all dimensions), border radius, border width, and colors
- Featured card highlighting with distinct background and text colors
- Responsive grid controls with independent column counts and gaps for desktop, tablet, and mobile
- Complete typography controls for all text elements (heading, subheading, plan name, price, features, button) with responsive font sizes
- Button customization including text, URL, target, colors, hover states, and glow effects with opacity control
- Yearly badge with customizable text, background color, and font size
- Container width controls with fullwidth/constrained modes and responsive max-width settings
- Card hover effects with customizable background and text colors
- Text and button alignment controls for card content

### Progress Block (Plus)
- Scroll-based progress indicator with two variants: bar and ring
- Bar variant: customizable length, width (vw units), fill color, track color, and border radius
- Ring variant: customizable size, stroke width, fill color, track color, text color, and font size
- Fixed positioning with flexible offset controls
- Vertical position selector (top/bottom) with percentage offset slider
- Horizontal position selector (left/right) with percentage offset slider
- Z-index control for proper layering
- Smooth scroll tracking with real-time progress updates

### Animation on Scroll Block (Free)
- Complete scroll-triggered animation system with 12 animation types
- Animation types: fade-in, fade-out, fade-left, fade-right, slide-up, slide-down, zoom-in, zoom-out, rotate, scale, blur, and bounce
- Configurable animation distance for fly/fade animations (0-300px)
- Flip axis controls (horizontal/vertical) and direction (clockwise/anti-clockwise)
- Customizable animation duration and delay
- Live editor preview for real-time animation testing
- Bidirectional reversal when scrolling in either direction
- Full inner blocks support for animating any content
- Threshold and delay controls for precise animation timing

### Social Share Block (Free)
- Social sharing button with customizable icon selection using Bootstrap Icons
- Tooltip interface displaying available sharing platforms
- Platform selection with enable/disable toggles for each service
- Supported platforms: Facebook, X (Twitter), LinkedIn, WhatsApp, Telegram, Reddit, Pinterest, and Email
- Customizable platform icons and labels using Bootstrap Icons
- Button styling controls including background color, border, border radius, padding, and icon color
- Tooltip positioning options (above/below button)
- Automatic page URL and title extraction for sharing
- One-click sharing functionality opening platform-specific share dialogs

### Testimonial(Premium)
- Premium testimonial carousel with draggable interface
- Purple background styling with white card design
- Profile image support with customizable sizing
- Responsive image controls for desktop, tablet, and mobile
- Keyboard-accessible navigation with arrow controls
- Unlimited testimonials with add/remove/reorder functionality
- Rich text support for testimonial quotes
- Customizable typography for name, title, and quote text
- Container width controls with responsive max-width settings

## Block Management Improvements

### Tier-Based Organization
- Blocks now organized by tier (Free, Plus, Premium) in the admin settings page
- Clear visual separation with section headers for each tier
- Improved navigation and block discovery experience

### Auxiliary Block Management
- Auxiliary blocks (child blocks like Content Toggle Panel, Swiper Slide, etc.) are now hidden from the block management interface
- Parent-child block synchronization ensures auxiliary blocks automatically follow their parent's enabled/disabled state
- Reduced interface clutter while maintaining full functionality

### Icon System Enhancements
- All blocks now use consistent SVG icons across editor, admin interface, and documentation
- Removed black fills from Flip Card, Our Process, and Swiper Carousel icons for cleaner appearance
- Fixed icon display issues in block management admin page
- Updated Animation on Scroll block icon to correct design
