# Adaire Blocks v1.1.5 Release Notes

## Particles Block Enhancement & User Experience Improvements

This release brings significant improvements to the Particles Block with enhanced text section controls and better visual alignment, along with improved admin interface icons for better block identification.

### Major Improvements

#### Particles Block Text Section Controls
- **Enhanced Layout Management**: Added comprehensive controls for text sections in Particles Block
- **Improved Alignment**: Better text positioning and alignment options for perfect visual coordination
- **Better Image-Text Coordination**: Text sections now properly align with their corresponding images
- **Responsive Design**: Text controls work seamlessly across all device sizes

#### Admin Interface Improvements
- **Custom Block Icons**: All blocks now display with custom SVG icons in the WordPress admin
- **Better Visual Identification**: Easier block recognition and management in the editor
- **Consistent Design**: Unified icon system across all Adaire Blocks
