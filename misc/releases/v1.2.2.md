# Adaire Blocks v1.2.2 Release Notes

Version 1.2.2 is a consistency and layout enhancement release that introduces new blocks across Free and Plus tiers, unifies layout settings across all infogrid blocks, and standardizes responsive breakpoints throughout the plugin.

## New Blocks

### Free Blocks
- **Hero 1** (`hero-1-block`) - A responsive hero block with configurable settings for all breakpoints, featuring flexible layout controls, responsive padding, and max-width settings
- **Info Grid** (`infogrid-block`) - A beautiful info grid with hover-expandable items featuring title, tagline, and description
- **Info Grid 2** (`infogrid-2-block`) - Advanced grid layout with customizable items, backgrounds, and overlay options
- **Image Composition** (`image-composition-block`) - Create collage-style compositions of images in an asymmetric grid layout

### Plus Blocks
- **Info Grid 3** (`infogrid-3-block`) - A responsive 2-row grid block with a three-column sub-section and background options
- **Info Grid 4** (`infogrid-4-block`) - A 2x2 grid block with configurable colors, typography, and background SVG icon support
- **Case Studies** (`case-studies-block`) - A dynamic case studies grid with GSAP FLIP animations, filtering by Industry and Capabilities, and load more functionality
- **Video Carousel** (`video-carousel-block`) - A draggable video carousel with custom drag cursor and smooth scrolling cards

## Layout Consistency Improvements

### Unified Layout Settings System
- **Consistent Controls**: All infogrid blocks (infogrid, infogrid2, infogrid3, infogrid4) now share identical layout settings panels
- **Standardized Interface**: Same controls for Container Width (Full Width/Constrained), Max Width, and Block Padding across all blocks
- **Device Selector**: Unified device selector with 5 breakpoints (Mobile, Tablet, Small Laptop, Desktop, Big Desktop) on all infogrid blocks
- **Real-time Preview**: Editor preview updates immediately when layout settings change

### Infogrid Block Updates
- **Added Layout Settings Panel**: Replaced old "Container Settings" with new "Layout Settings" panel matching infogrid3/infogrid4
- **Added Responsive Padding**: Full responsive padding controls (top, right, bottom, left) for all 5 breakpoints
- **Enhanced Max Width Controls**: RangeControl with unit selection (px, %, rem, vw) matching other blocks
- **Device Toggle Integration**: Added device selector with custom icons for all breakpoints
- **CSS Variable System**: Implemented responsive padding CSS variables for all breakpoints

### Infogrid2 Block Updates
- **Replaced Grid Layout Panel**: Removed old "Grid Layout" panel, replaced with "Layout Settings" panel
- **Added Container Mode**: Full Width/Constrained toggle for flexible container control
- **Added Max Width Controls**: Responsive max-width settings with RangeControl (max 10000px) and unit selection
- **Added Block Padding**: BoxControl for responsive padding on all 5 breakpoints
- **Fixed Visual Issues**: Removed black outline/border issues by consolidating border properties and adding `!important` overrides
- **Consistent Breakpoints**: Aligned breakpoint values with hero1 block as source of truth

### Call to Action Block Updates
- **Added Layout Settings**: Implemented same layout controls as infogrid blocks
- **Added Small Laptop & Big Desktop**: Extended breakpoint support from 3 to 5 breakpoints
- **Real-time Editor Preview**: Fixed constrained/full-width settings to update in editor preview
- **Responsive Padding**: Added responsive padding controls matching infogrid blocks
- **Container Max Width**: Enhanced with all 5 breakpoints for consistent responsive behavior

## Breakpoint Standardization

### Unified Breakpoint System
All blocks now use consistent breakpoint definitions based on hero1 block as the source of truth:
- **Mobile**: `max-width: 600px`
- **Tablet**: `min-width: 601px and max-width: 1024px`
- **Small Laptop**: `min-width: 1025px and max-width: 1300px`
- **Desktop**: `min-width: 1301px and max-width: 1920px`
- **Big Desktop**: `min-width: 1921px`

## Technical Improvements

### Editor Experience
- **Device Toggle Component**: Standardized device selector with custom icons for small-laptop and big-desktop breakpoints
- **Helper Functions**: Added `formatDimensionValue` and `updateContainerDimension` for consistent dimension handling
- **CSS Variable Naming**: Consistent naming convention for responsive CSS variables across all blocks

### Code Quality
- **Removed Duplicate Imports**: Fixed duplicate `RangeControl` import in call-to-action-block
- **Moved Utility Functions**: Extracted `formatDimensionValue` outside component scope to prevent dependency issues
- **Consistent Attribute Structure**: All blocks use same attribute structure for `responsiveMaxWidth` and `responsivePadding`
- **Proper CSS Fallbacks**: Responsive padding CSS variables with fallback chain (big-desktop → desktop → small-laptop → tablet → mobile)

### Validation Server Update
- **Domain Migration**: Updated all validation server references from `adairedigital.com` to `adaireblocks.com`
- **License API Endpoint**: Updated to `https://adaireblocks.com/wp-json/lmfwc/v2/licenses/`

- **Documentation**: Updated validation server documentation with new domain
