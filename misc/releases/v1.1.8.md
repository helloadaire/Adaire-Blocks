# Adaire Blocks v1.1.8 Release Notes

## New Blocks - Modal Block & Industries Block

This release introduces two powerful new blocks: the **Modal Block**, a flexible modal dialog system, and the **Industries Block**, a comprehensive solution for showcasing industries with responsive tiles, icons, and rich content. Both blocks feature extensive customization options and responsive design.

### Modal Block - Complete Modal Dialog System

The **Modal Block** provides a complete solution for creating modal dialogs with complete control over appearance, dimensions, and behavior. It features a flexible inner blocks system that allows you to add any Gutenberg blocks to both the trigger and content areas.

### Major New Features

#### Modal Block Architecture
- **Two-Block System**: Separate trigger and content blocks for maximum flexibility
- **Flexible Inner Blocks**: Add any Gutenberg blocks to trigger and content areas
- **Inline Display**: Modal block displays inline to minimize obstruction when adding to pages
- **Template System**: Default templates for quick setup (button trigger, heading + paragraph content)
- **Template Lock**: Prevents accidental removal of essential inner blocks

#### Responsive Dimension Controls
- **Device-Specific Settings**: Separate width and height controls for desktop, tablet, and mobile
- **Multiple Unit Support**: 
  - Width: px, %, vw, rem
  - Height: px, %, vh, auto
- **Real-Time Preview**: See dimension changes immediately in the editor
- **Responsive Defaults**: 
  - Desktop: 600px width, 400px height
  - Tablet: 90% width, 400px height
  - Mobile: 90% width, 360px height
- **Content-Based Sizing**: Modal container uses fit-content to wrap content properly

#### Comprehensive Styling Options
- **Background Color**: Full color picker with alpha channel support
- **Overlay Color**: Customizable backdrop overlay with transparency control
- **Border Customization**: Color, width (0-12px), and radius (0-48px) controls
- **Padding Control**: Adjustable padding (0-80px) with responsive clamping on mobile
- **Close Button**: Customizable size (24-64px), icon color, and background color
- **Smooth Animations**: CSS transitions for opening and closing animations

#### Accessibility Features
- **Keyboard Navigation**: Full keyboard support (Enter, Space, Escape, Tab)
- **Focus Management**: Automatic focus movement when modal opens/closes
- **Focus Trap**: Tab navigation trapped within modal when open
- **ARIA Attributes**: Proper roles and labels for screen readers
- **Semantic HTML**: Uses proper button and dialog elements
- **Body Scroll Lock**: Page scrolling disabled when modal is open

### Detailed Features

- **Desktop Controls**: Configure width and height for desktop screens
- **Tablet Controls**: Responsive settings for tablet devices (≤ 1024px)
- **Mobile Controls**: Optimized settings for mobile devices (≤ 768px)

#### Content Management

### Industries Block - Industry Showcase System

The **Industries Block** provides a complete solution for displaying industries with responsive tiles, customizable icons, and flexible layouts. It features comprehensive management controls, link functionality, and smart visibility options.

#### Industries Block Architecture
- **Dynamic Industry Management**: Add, remove, and reorder industries with intuitive controls
- **Icon Picker Integration**: Visual Bootstrap Icons picker for easy icon selection
- **Link Functionality**: Add URLs to each industry with option to open in new tab
- **Responsive Layouts**: Separate mobile grid and desktop layouts optimized for each screen size
- **Flexible Display Options**: Option to hide first two industries from layout
- **Visibility Controls**: Toggle title, intro text, and image visibility with smart space management

#### Industry Management Features
- **Add Industries**: Easily add new industries with default icon and label
- **Remove Industries**: Remove individual industries with one click
- **Reorder Industries**: Move industries up or down to customize order
- **Icon Selection**: Visual icon picker with searchable Bootstrap Icons library
- **Icon Removal**: Quick option to remove icons from industries
- **Link Management**: Add URLs to make industries clickable with new tab option

#### Responsive Layout System
- **Mobile Grid**: All industries displayed in responsive grid on mobile/tablet
- **Desktop Layout**: 
  - First two industries displayed as large squares (optional)
  - Remaining industries displayed in two-column grid
  - Dynamic distribution adapts to any number of industries
- **Layout Flexibility**: Option to hide first two industries, starting display from third item
- **Smart Distribution**: Industries automatically distributed into columns for optimal display

#### Visibility & Space Management
- **Title Visibility**: Toggle title visibility with space preservation on desktop (≥1200px)
- **Intro Text Visibility**: Control intro text display with responsive space management
- **Image Visibility**: Toggle feature image with smart layout preservation
- **Responsive Behavior**: 
  - Desktop (≥1200px): Hidden elements preserve space using `visibility: hidden`
  - Mobile (<1200px): Hidden elements collapse space using `display: none`
- **Layout Preservation**: Desktop layout remains intact when elements are hidden

#### Styling & Customization
- **Container Controls**: Full-width or constrained container modes
- **Responsive Max-Width**: Device-specific max-width controls (desktop, tablet, mobile)
- **Typography Controls**: Customizable font sizes for headers, intro text, and descriptions
- **Color Customization**: Background, header colors, accent colors, and tile colors
- **Tile Styling**: Padding, border radius, border color, icon colors and backgrounds
- **Hover Effects**: Customizable hover background and text colors
- **Chevron Customization**: Size and color controls for navigation indicators

#### Content Management
- **Header Lines**: Multiple headline support with customizable text
- **Intro Text**: Rich text intro section with visibility control
- **Description Text**: Detailed description area with visibility toggle
- **Feature Image**: Image upload with alt text support and visibility control
- **Industry Items**: Each industry includes icon, label, and optional link

#### Block Attributes

**Modal Block:**
- `blockId`: Unique identifier for the modal block
- `modalWidth`: Responsive width object (desktop, tablet, mobile with value/unit)
- `modalHeight`: Responsive height object (desktop, tablet, mobile with value/unit)
- `backgroundColor`: Modal background color
- `overlayColor`: Overlay backdrop color
- `borderColor`: Border color
- `borderWidth`: Border width in pixels (0-12)
- `borderRadius`: Border radius in pixels (0-48)
- `padding`: Internal padding in pixels (0-80)
- `closeButtonColor`: Close button icon color
- `closeButtonBackground`: Close button background color
- `closeButtonSize`: Close button size in pixels (24-64)

**Modal Trigger Block:**
- Uses Inner Blocks API for content
- Default template: Button block
- No custom attributes (content-based)

**Modal Content Block:**
- Uses Inner Blocks API for content
- Default template: Heading + Paragraph
- No custom attributes (content-based)

**Industries Block:**
- `blockId`: Unique identifier for the block
- `containerMode`: Full-width or constrained container ("full" or "constrained")
- `containerMaxWidth`: Responsive max-width object with desktop/tablet/mobile (value/unit)
- `headerLines`: Array of headline objects with id and text properties
- `introText`: Intro text string
- `descriptionText`: Description text string
- `featureImage`: Image object with id, url, and alt properties
- `showTitle`: Boolean to toggle title visibility (default: true)
- `showIntroText`: Boolean to toggle intro text visibility (default: true)
- `showImage`: Boolean to toggle image visibility (default: true)
- `hideFirstTwoIndustries`: Boolean to hide first two industries from layout (default: false)
- `industries`: Array of industry objects with:
  - `id`: Unique identifier
  - `icon`: Bootstrap Icon class string (e.g., "bi bi-basket2")
  - `label`: Industry label text
  - `linkUrl`: Optional URL string
  - `openInNewTab`: Boolean for link target (default: false)
- `backgroundColor`: Background color hex string
- `headerColor`: Header text color hex string
- `headerAccentColor`: Header accent color hex string
- `headerFontWeight`: Header font weight string
- `headerFontSize`: Header font size number (pixels)
- `introFontSize`: Intro text font size number (pixels)
- `descriptionFontSize`: Description font size number (pixels)
- `chevronColor`: Chevron icon color hex string
- `chevronSize`: Chevron icon size number (pixels)
- `tilePadding`: Tile padding number (pixels, desktop)
- `tilePaddingMobile`: Tile padding number (pixels, mobile)
- `tileBorderRadius`: Tile border radius number (pixels)
- `tileBorderColor`: Tile border color hex string
- `tileIconColor`: Icon color hex string
- `tileIconBackground`: Icon background color hex string
- `tileIconSize`: Icon size number (pixels)
- `tileTitleFontSize`: Industry title font size number (pixels)
- `tileHoverBackground`: Hover background color hex string
- `tileHoverTextColor`: Hover text color hex string

### Getting Started

#### Modal Block

1. **Add the Block**: Insert "Modal" from the block inserter
2. **Customize Trigger**: Edit the trigger block (default: button) or add your own content
3. **Add Modal Content**: Edit the content block (default: heading + paragraph) or add your own blocks
4. **Set Dimensions**: Configure width and height for desktop, tablet, and mobile
5. **Customize Styling**: Adjust colors, borders, padding, and close button
6. **Preview**: Use preview toggle to see modal appearance in editor
7. **Test**: Click trigger on frontend to open modal and test functionality

#### Industries Block

1. **Add the Block**: Insert "Industries" from the block inserter
2. **Edit Headlines**: Customize the header lines and intro text
3. **Add Industries**: Use "Add Industry" button to add new industries
4. **Select Icons**: Click icon picker to choose Bootstrap Icons for each industry
5. **Add Links**: Optionally add URLs to make industries clickable
6. **Reorder Industries**: Use move up/down buttons to reorder industries
7. **Customize Layout**: Toggle visibility of title, intro text, and image
8. **Hide First Two**: Optionally hide first two industries from layout
9. **Style Customization**: Adjust colors, typography, and tile styling
10. **Set Container**: Choose full-width or constrained container mode

#### Best Practices

- **Content Planning**: Plan modal content before setting dimensions

- **Performance**: Keep modal content lightweight

- **Color Contrast**: Ensure sufficient contrast for readability
