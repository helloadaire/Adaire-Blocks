# Adaire Blocks v1.1.9 – New Storytelling Blocks & Fresh Icons 🎉

This update brings four brand-new blocks plus a complete icon refresh across Free, Plus, and Premium tiers.

## ✨ New Blocks

### 🧭 Our Process Block
Showcase your delivery process with beautiful step tiles. Set different column counts for desktop/tablet/mobile, pick icons, add optional links, and control every color/hover state.

### 🔁 Content Toggle Block
Let visitors switch between content panels using pill-style toggles. Customize active/inactive colors, typography, placement (top/bottom), animation speed/ease, and responsive container widths.

### 🖼️ Swiper Carousel + Swiper Slide
Build draggable Swiper.js carousels right in Gutenberg. Configure autoplay, loop, speed, effects, cursor overlay text/color, and design each slide with media, gradients, and copy.

### 💬 Testimonial 2 Block
Hero-style testimonials with large portraits, company logos, and keyboard-friendly nav arrows. Tune background/card colors, typography, and responsive image sizes for every breakpoint.

## 🎨 Icon Refresh
Every block now ships with a crisp SVG icon drawn from `src/icons/new-icons`. Gutenberg, the block inserter, the block management page, and the readmes all display the same polished artwork (no more black squares).

## Getting Started
All new blocks live in the inserter (search for “Process”, “Content Toggle”, “Swiper”, or “Testimonial 2”). Drop them into any page, tweak the settings panel, and you’re done—no custom code required.

## Why Upgrade?
- More storytelling blocks for process pages, product explainers, and testimonials.
- Built-in Swiper carousel so you don’t need custom HTML.
- Unified icon language for a more professional editor experience.

No migrations or breaking changes—just update and start using the new blocks. 🚀
