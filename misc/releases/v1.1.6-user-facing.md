# Adaire Blocks v1.1.6 Release Notes

## Enhanced Mega Menu Block & Improved Build Stability

This release brings significant improvements to the Mega Menu Block with enhanced color controls and better menu item styling, along with improved build process stability for more reliable plugin generation.

- **Better Color Inheritance**: Menu items now properly inherit and display colors as intended
- **Visual Consistency**: Enhanced color control interface with improved organization and usability

#### Build Process Improvements
- **Stable Generation**: Fixed free plugin generation process with improved error handling
- **Better Compatibility**: Optimized build process for better compatibility across WordPress environments
- **Reliable Builds**: Enhanced build stability for more consistent plugin generation
