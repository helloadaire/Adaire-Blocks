# Adaire Blocks v1.1.9 Release Notes

Version 1.1.9 is a feature release that adds four brand-new storytelling blocks and refreshes every icon in the plugin so the editor, block management UI, and documentation all share the same modern artwork.

## New Blocks

### Our Process Block
- Responsive multi-column process layout with independent column counts and gaps for desktop, tablet, and mobile.
- Per-step icon picker (Bootstrap Icons), drag-to-reorder controls, and optional link target so each step can point to `/something`.
- Device-aware container modes (full or constrained) plus max-width sliders for every breakpoint.
- Complete styling controls: headline/subheadline typography, box padding, border radius/width, hover states, and individual color tokens for icons and text.

### Content Toggle Block
- Pill-style toggle navigation that lets visitors flip between multiple rich content panels without page reloads.
- GSAP-powered transitions with adjustable duration/ease, pill placement (top or bottom), and active/inactive color systems.
- Toggle management UI with add/remove/reorder actions, per-pill labels, and an initial active toggle selector.
- Container width controls, content padding/radius settings, and typography options for pills and body copy.

### Swiper Carousel & Swiper Slide Blocks
- Native Swiper.js integration with draggable slides, optional autoplay, loop, speed, and effect (slide/fade/etc.) controls.
- Carousel height, global border radius, and draggable cursor overlay (color + label) for better UX hints.
- Swiper Slide child block supplies media management (WP media picker), overlay type (solid or gradient), and per-slide copy (header, paragraph) with color controls.
- Pairing Carousel + Slide blocks lets you build case studies, hero sliders, and visual teasers without leaving the block editor.

### Testimonial 2 Block
- Hero-style testimonial layout with large portrait photography, floating company logo, and pill-shaped quote mark overlay.
- Keyboard-accessible carousel navigation with configurable button size, hover states, and arrow icon size.
- Full typography stack (description, name, title), card/background colors, logo sizing, and responsive image sizing per device.
- Supports unlimited testimonials with reorder and edit-in-place controls.

## Icon System Refresh
- Every Free, Plus, and Premium block received a brand-new SVG icon.
- Icons are now stored as standalone components sourced from `src/icons/new-icons` and injected automatically into each `block.json`.
- Gutenberg, the block inserter, the block management screen, and both readme files all display consistent, full-color icons (no more black boxes).

## Developer Notes
- `scripts/apply-new-icons.js` now parses `.svg` files, inlines `class` styles as real SVG attributes, and rewrites the icon components.
- `scripts/update-block-icons.js` continues to run automatically via `npm run prebuild`, guaranteeing block metadata always embeds the latest artwork.
- Added block documentation for Our Process, Content Toggle, Swiper Carousel/Slide, and Testimonial 2.
- Updated testing notes, release notes, update JSON, and readme files to reflect the new functionality.

## Upgrade Impact
- No breaking changes—existing blocks keep working.
- New blocks are instantly available in the inserter after updating.
- If you maintain custom block allowlists, add:
  - `create-block/our-process-block`
  - `create-block/content-toggle-block`
  - `adaire/swiper-carousel` and `adaire/swiper-slide`
  - `create-block/testimonial2-block`

Enjoy building with the new storytelling blocks and refreshed icons!
