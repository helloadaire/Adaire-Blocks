# Adaire Blocks v1.2.4 – New Block Icons & Better Block Picker

## What's New

**Custom icons for more blocks**  
Hero 1, Image Composition, Info Grid, Info Grid 2, Info Grid 3, Info Grid 4, Card Scroll, Card Scroll Item, and Footer blocks now use custom icons everywhere: in the block picker when you add a block, and in the block management screen.

**Block picker looks right**  
Those same blocks now show their icons correctly in the block inserter (block picker), so it’s easier to spot the block you want.

**Free plugin: blocks are categorized**  
In the free version, all free blocks now appear under **“ADAIRE BLOCKS FREE”** in the block picker instead of “Uncategorized,” so they’re easier to find.

## For Users

- **No action required** – Update the plugin and you’ll get the new icons and (on the free build) the new category automatically.
- **Visual consistency** – Icons match across the editor and block management.
- **Easier discovery** – Free users see their blocks in one place under “ADAIRE BLOCKS FREE.”

## Technical Summary

- New SVG assets added and wired through the existing icon system (part5 and Free Blocks).
- Block registration updated to use React icon components so the block picker can display icons reliably.
- Free build script updated to register the “ADAIRE BLOCKS FREE” block category.
