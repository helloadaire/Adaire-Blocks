# Adaire Blocks v1.2.2 – Layout Consistency Update

This release introduces new blocks across Free and Plus tiers, unifies layout controls across all infogrid blocks, and improves the consistency of responsive design settings throughout the plugin.

## New Blocks

### Free Blocks
- **Hero 1** - A responsive hero block with flexible layout controls and responsive padding
- **Info Grid** - Beautiful info grid with hover-expandable items
- **Info Grid 2** - Advanced grid layout with customizable backgrounds and overlays
- **Image Composition** - Create collage-style image compositions in asymmetric grid layouts

### Plus Blocks
- **Info Grid 3** - 2-row grid block with three-column sub-section
- **Info Grid 4** - 2x2 grid block with configurable colors and typography
- **Case Studies** - Dynamic case studies grid with filtering and animations
- **Video Carousel** - Draggable video carousel with smooth scrolling

## What's Improved

### Unified Layout Controls
All infogrid blocks (infogrid, infogrid2, infogrid3, infogrid4) now share the same layout settings panel, making it easier to configure containers, max-widths, and padding consistently across your blocks.

### Standardized Breakpoints
The plugin now uses a unified 5-breakpoint system (Mobile, Tablet, Small Laptop, Desktop, Big Desktop) across all blocks, ensuring consistent responsive behavior throughout your site.

### Enhanced Call to Action Block
The Call to Action block now includes the same powerful layout controls as infogrid blocks, with responsive padding and max-width settings that update in real-time in the editor.

### Visual Improvements
Fixed visual outline issues in the infogrid2 block for a cleaner, more professional appearance.

## For Users

### Better Consistency
- All infogrid blocks now work the same way
- Layout controls are in the same place across all blocks
- Easier to learn and use

### More Control
- Responsive padding controls for all breakpoints
- Flexible container width options (Full Width/Constrained)
- Max-width settings with multiple unit options (px, %, rem, vw)

### Real-time Preview
Changes to layout settings now update immediately in the editor preview, so you can see exactly how your blocks will look before publishing.

## Technical Updates

- Standardized breakpoint system across all blocks
- Unified layout settings interface
- Improved responsive padding controls
- Enhanced editor preview functionality
- Updated validation server endpoints

No action required—just update and enjoy the improved layout consistency!
