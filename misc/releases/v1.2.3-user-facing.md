# Adaire Blocks v1.2.3 – Enhanced Carousels & Mobile Experience

This release brings smoother carousel functionality, enhanced responsive controls, and better mobile interactions across multiple blocks.

## What's Improved

### Smoother Carousels
- **Video Carousel Block**: Now uses Swiper.js for smoother scrolling and better performance
- **Swiper Carousel Block**: Complete overhaul with Swiper.js, matching Video Carousel functionality
- **Better Mobile Experience**: Improved touch interactions and smoother scrolling on mobile devices
- **Enhanced Controls**: More styling options with full breakpoint support

### Better Mobile Interactions
- **Flip Card Block**: Cards now flip back when tapped on mobile and tablet devices
- **Touch-Friendly**: Automatic detection of touch devices for better user experience

### Enhanced Modal Block
- **Full Responsive Control**: Set width and height for all screen sizes (mobile, tablet, small laptop, desktop, big desktop)
- **Better Positioning**: Improved z-index to ensure modals appear above navigation

### Social Banner Block Enhancements
- **More Positioning Options**: Choose between fixed, absolute, or relative positioning
- **Horizontal Offsets**: Set left/right offsets just like top/bottom offsets
- **Background Styling**: Customize background color and border radius
- **Responsive Padding**: Control padding for each side at every breakpoint

## For Users

### Better Performance
- Carousels now scroll more smoothly and respond better to touch interactions
- Reduced jitter and improved stability on mobile devices

### More Control
- Set carousel slide heights with different units (pixels, percentages, rem, vw)
- Control pagination visibility and colors
- Customize modal dimensions for every screen size
- Fine-tune social banner positioning and styling

### Better Mobile Experience
- Flip cards work naturally on touch devices
- Carousels respond better to touch gestures
- All responsive controls work seamlessly across devices

## Technical Updates

- Migrated Video Carousel and Swiper Carousel to Swiper.js
- Added comprehensive responsive controls with full breakpoint support
- Enhanced touch interaction support
- Improved real-time editor preview updates
- Fixed various editor preview issues

No action required—just update and enjoy the smoother carousels and enhanced controls!
