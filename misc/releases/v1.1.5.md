# Adaire Blocks v1.1.5 Release Notes

## Free Version Generator Overhaul & Packaging Compliance

This release focuses on generating a clean WordPress.org–compliant FREE build, removing all license system dependencies, stabilizing builds, and improving admin settings icon rendering.

### Major Improvements

#### Free Version Generator (WordPress.org–compliant)
- Removed all license-related files and code from the generated FREE plugin
- Stripped license includes, notices, and checks from `adaire-blocks.php`
- Cleaned license checks in `includes/class-adaire-blocks-config.php`
- Excluded `plugin-update-checker/` and rollback logic from FREE build

#### Build Stabilization for FREE
- Removed `prebuild` icons updater from FREE `package.json` to avoid missing script errors
- Ensured `wp-scripts build --blocks-manifest` runs for enabled blocks only
- Warnings only; no fatal build errors

#### Admin Settings Page Icons
- FREE settings page now renders exact SVGs from `src/icons/icon-svgs/{block}.svg`
- Dashicon fallback when an SVG file is not present

#### Particles Block Text Section Controls
- Added comprehensive controls for text sections in Particles Block
- Improved text alignment and positioning options
- Enhanced layout management for better text-image coordination

### Technical Improvements

#### Generator Script Enhancements
- Added directory/file filters to exclude license-related assets
- Regex-based cleanup to remove license/update code from `adaire-blocks.php`
- Post-copy cleanup of `admin/` and `includes/` directories
- Wrote FREE `config/blocks-config.json` with `version: "free"` and empty `premium {}`

#### Verification & Safety Nets
- Automatic verification logs after generation (icons present, config flags, update code removed)
- Build continues even if optional steps fail (with clear warnings)

### Bug Fixes
- Fixed fatal error on activation in FREE builds caused by missing `class-adaire-blocks-license.php`
- Removed lingering license notices and references that could trigger admin warnings
- Prevented missing `scripts/update-block-icons.js` from breaking FREE builds

### What's Fixed
- FREE package no longer references license classes or pages
- FREE `adaire-blocks.php` is clean of Plugin Update Checker and rollback code
- Admin settings page correctly uses exact SVG icons where available
- Particles Block text sections now have proper alignment and positioning controls

### Benefits for Users
- FREE build installs and activates without errors
- Clear, consistent admin visuals with SVG icons
- WordPress.org–ready packaging with no third-party update systems

### Technical Details

#### Files Modified
- `scripts/generate-free-version.js` – copy filters, cleanup routines, package.json adjustments, verification
- `admin/settings-page.php` (FREE output) – SVG rendering logic
- `includes/class-adaire-blocks-config.php` (FREE output) – removed license checks
- `adaire-blocks.php` (FREE output) – removed license, update, and rollback code

#### Key Changes
- Exclude: `admin/license-page.php`, `admin/js/license-page.js`, `admin/css/license-page.css`
- Exclude: `includes/class-adaire-blocks-license.php`, `includes/class-adaire-blocks-validation-config.php`
- Exclude: `plugin-update-checker/` directory
- Sanitized FREE `package.json` (no `prebuild`)

### Migration Notes
- No action required; changes affect FREE packaging only
- Premium builds are unaffected
