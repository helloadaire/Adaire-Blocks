# Adaire Blocks v1.2.3 Release Notes

Version 1.2.3 is a functionality and interaction enhancement release that improves carousel performance, adds comprehensive styling controls, enhances modal responsiveness, and introduces touch interactions for better mobile experience.

## Summary

This release focuses on improving user experience through better carousel mechanics, enhanced responsive controls, and mobile-friendly interactions. The Video Carousel and Swiper Carousel blocks have been migrated to Swiper.js for smoother performance, the Modal block now supports full responsive dimensions, the Flip Card block supports touch interactions, and the Social Banner block has been significantly enhanced with advanced positioning and styling options.

## New Features

### Video Carousel Block Enhancements
- **Swiper.js Migration**: Migrated from Splide.js to Swiper.js for improved carousel functionality
- **Smoother Performance**: Better scrolling mechanics and smoother transitions
- **Improved Loop Support**: Enhanced bidirectional looping functionality
- **Mobile Peek Support**: Better handling of adjacent card visibility on mobile devices
- **Centered Slides**: Automatic centering when slides per view is set to 1
- **Drag Cursor Animation**: Enhanced drag cursor with GSAP animations

### Swiper Carousel Block Overhaul
- **Complete Swiper.js Integration**: Full migration to Swiper.js matching Video Carousel mechanics
- **Responsive Slide Height**: Added height controls with unit options (px, %, rem, vw) for all breakpoints
- **Pagination Controls**: Added visibility toggle and color controls for active/inactive pagination dots
- **Comprehensive Styling**: Full breakpoint support for all styling controls
- **Drag Indicator**: Added drag cursor with customizable text, size, colors, and styling
- **Container Controls**: Added container mode and max-width controls matching Video Carousel
- **List Padding**: Added responsive padding-left controls for all breakpoints

### Modal Block Enhancements
- **Full Breakpoint Support**: Added responsive width and height controls for all 5 breakpoints
- **Removed Percentage Width**: Removed percentage option for width as modal is freely floating
- **Consistent Breakpoints**: Aligned with standard breakpoint system (mobile, tablet, small-laptop, desktop, big-desktop)
- **Enhanced Z-index**: Increased z-index to ensure modal appears above navbar

### Flip Card Block Enhancements
- **Touch Interaction Support**: Added touch-responsive functionality for mobile and tablet devices
- **Flip Back on Tap**: Cards can now flip back when tapped again on touch devices
- **Touch Device Detection**: Automatic detection of touch devices with CSS class application
- **Smooth Animations**: Maintained smooth 3D transitions for touch interactions

### Social Banner Block Major Enhancements
- **Position Type Options**: Added fixed, absolute, and relative positioning options
- **Display Type Control**: Added block and inline display options for easier positioning
- **Horizontal Offset Controls**: Added left/right offset controls matching vertical offset functionality
- **Background Color Control**: Added color picker for background color customization
- **Responsive Padding**: Added comprehensive padding controls (top, right, bottom, left) for all 5 breakpoints
- **Background Border Radius**: Added border radius control for background styling
- **Real-time Preview**: Enhanced editor preview with real-time updates for all background settings

## Technical Improvements

### Carousel Mechanics
- **Unified Carousel Library**: Both Video Carousel and Swiper Carousel now use Swiper.js for consistency
- **Improved Loop Logic**: Enhanced loop calculation with `loopedSlides` based on max slides per view
- **Better Mobile Handling**: Improved mobile peek functionality with centered slides support
- **Smooth Scrolling**: Enhanced momentum scrolling and drag behavior
- **Observer Support**: Added Swiper observer settings for dynamic content updates

### Editor Experience
- **Real-time Updates**: Improved real-time preview updates for all responsive settings
- **Breakpoint Consistency**: All blocks now use consistent 5-breakpoint system
- **CSS Variable System**: Enhanced CSS variable usage for responsive styling
- **Editor Styles**: Improved editor preview styling for better accuracy

### Code Quality
- **Fixed React Errors**: Resolved React error #130 in Social Banner Block by correcting PanelColorSettings import
- **Import Corrections**: Fixed component imports to use correct WordPress packages
- **CSS Variable Handling**: Improved handling of empty color values with proper fallbacks
- **Breakpoint Mixins**: Standardized breakpoint mixins across all blocks

## Upgrade Impact

### Breaking Changes
- None. All changes are backward compatible.

### Migration Notes
- Video Carousel blocks will automatically migrate from Splide.js to Swiper.js markup
- Swiper Carousel blocks will automatically migrate from Splide.js to Swiper.js markup
- Flip Card blocks will automatically gain touch interaction support
- Social Banner blocks will maintain existing positioning but gain new options

### Backward Compatibility
- All existing blocks remain fully functional
- Auto-recovery system handles markup changes automatically
- Deprecated save versions ensure smooth transitions

## Performance Considerations

### Improvements
- **Swiper.js Performance**: Swiper.js provides better performance than Splide.js for carousel functionality
- **Reduced JavaScript**: Unified carousel library reduces overall JavaScript footprint
- **Optimized Animations**: Enhanced GSAP animations for drag cursor with better performance
- **CSS Variable Efficiency**: Improved CSS variable usage reduces style recalculation

### Considerations
- Swiper.js is slightly larger than Splide.js but provides better functionality
- Touch interaction detection adds minimal overhead
- Responsive padding controls add CSS variables but with minimal performance impact
