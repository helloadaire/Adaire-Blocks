### Testing Notes - v1.1.7 Release Verification

#### Overview

This document verifies the v1.1.7 release which introduced three major new blocks: Animation on Scroll Block, Video Player Block, and Map Block. This testing ensures all new features work correctly, animations function properly, and the blocks integrate seamlessly with existing functionality.

#### New Features Introduced (v1.1.7)

1) Animation on Scroll Block

- Complete control over scroll-triggered animations with 12 animation types
- Configurable animation distance for fly/fade animations (0-300px)
- Flip axis controls (horizontal/vertical) and direction (clockwise/anti-clockwise)
- Rotate animation type with customizable easing and duration
- Live editor preview for real-time animation testing
- Bidirectional reversal when scrolling in either direction

2) Video Player Block

- YouTube and Vimeo video embedding with automatic ID extraction from various URL formats
- Video playback controls (autoplay, mute, controls, loop)
- Customizable container height and border radius
- Smart URL parsing supporting multiple URL formats

3) Map Block

- Multiple location support with country, address, phone, and email
- Leaflet.js integration for smooth map fly animations between locations
- Coordinate-based map navigation with latitude/longitude support
- Customizable map appearance (navigation colors, font weight, fly speed)
- Two display modes (single map fly mode or multiple maps)

#### Fixes Implemented (v1.1.7)

1) Animation on Scroll Block fixes

- Fixed threshold and delay interaction conflicts with smart race condition prevention
- Fixed flip perspective to prevent element overflow on page
- Fixed fade-left and fade-right reversal behavior
