### Testing Notes - v1.1.4 Free Generation Verification

#### Overview

This document verifies the v1.1.4 changes to the Free version generator. It ensures all license-related code/files are removed, builds succeed, and the generated plugin activates and works in WordPress.

#### Fixes Implemented

1) License system removal in FREE output

- Removed copying of license-related files from `admin/` and `includes/`.
- Stripped all license code, notices, and includes from `adaire-blocks.php`.
- Cleaned license checks from `includes/class-adaire-blocks-config.php`.
- Key: `scripts/generate-free-version.js` (copy filters, cleanup methods, regex removals).

2) Update checker/rollback removal

- Removed Plugin Update Checker and rollback code from `adaire-blocks.php` in FREE build.
- Ensures WordPress.org-compliant update behavior.

3) Build script stabilization for FREE

- Removed `prebuild` script (icons updater) from FREE `package.json` to prevent missing script errors.
- Ensured `wp-scripts build --blocks-manifest` runs cleanly for enabled blocks only.

4) Admin settings SVG icons

- Updated `admin/settings-page.php` in FREE to render exact SVGs from `src/icons/icon-svgs/{block}.svg` with dashicon fallback.

5) Configuration adjustments

- FREE `config/blocks-config.json` gets `version: "free"` and `premium: {}`.
- Only enabled blocks are copied and registered.
